# /feature — Start a new feature
## Usage: /feature [app] [name] e.g. /feature web sale-void

1. Confirm app: web / mobile-api / admin
2. UltraThink — plan all files before any code
3. Show plan. Wait for approval.
4. Build in order: Migration → Model → Request → Service → Controller → Route → View/Resource
5. Run /review [app] before marking done
